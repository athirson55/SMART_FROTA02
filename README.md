# Smart Frota PI

Tela de login implementada em React + Vite com base no design do Figma.

## Stack

- React
- Vite
- React Router DOM
- Axios
- CSS puro

## Como executar

```bash
npm install
npm run dev
```

## Integracao com backend

Defina a variavel de ambiente `VITE_API_BASE_URL` em um arquivo `.env` para apontar para sua API.

Exemplo:

```bash
VITE_API_BASE_URL=http://localhost:3000
```
